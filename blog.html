Question-1: What are the different ways to select an element in the DOM?

Answer: The different ways to select an element in the DOM are getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: The difference between innerHTML, innerText, and textContent is
innerHTML: Parses HTML, including tags.
innerText: Shows rendered, visible text, respecting CSS.
textContent: Returns all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Answer: Event delegation in the DOM is a technique where we attach a single event listener to a parent element, rather than adding listeners to each of its child elements

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is when an event on a child element propagates up to its parent elements in the DOM tree, triggering the same event on those parents.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: I create, add, and remove elements using JavaScript with this code createElement() to create, appendChild() or insertBefore() to add, and removeChild() or remove() to remove HTML elements from the DOM.
